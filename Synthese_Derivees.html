DÉRIVÉES
Formules · Règles · Fonctions composées · Tangente · Optimisation
D1

Formules de dérivées de base

Ces formules doivent être connues par cœur — elles sont la fondation de tout le reste.

Fonction \(f(x)\)Dérivée \(f'(x)\)Exemple
\(C\) (constante) \(0\) \((5)' = 0\)
\(x\) \(1\) \((x)' = 1\)
\(x^n\) \(n \cdot x^{n-1}\) \((x^4)' = 4x^3\)
\(\sqrt{x} = x^{1/2}\) \(\dfrac{1}{2\sqrt{x}}\) \((\sqrt{x})' = \dfrac{1}{2\sqrt{x}}\)
\(\dfrac{1}{x} = x^{-1}\) \(-\dfrac{1}{x^2}\) \(\left(\dfrac{1}{x}\right)' = -\dfrac{1}{x^2}\)
\(\sin(x)\) \(\cos(x)\) \((\sin x)' = \cos x\)
\(\cos(x)\) \(-\sin(x)\) \((\cos x)' = -\sin x\)
\(\tan(x)\) \(\dfrac{1}{\cos^2(x)}\) \((\tan x)' = \dfrac{1}{\cos^2 x}\)
\(e^x\) \(e^x\) \((e^x)' = e^x\) (se dérive seul !)
\(\ln(x)\) \(\dfrac{1}{x}\) \((\ln x)' = \dfrac{1}{x}\)
💡Rappels utiles : \(\sqrt{x} = x^{1/2}\), \(\dfrac{1}{x} = x^{-1}\), \(\sqrt[3]{x} = x^{1/3}\). Comme ça tu peux toujours utiliser la règle \((x^n)' = nx^{n-1}\).
D2

Règles de dérivation

Somme / Différence
\[(f + g)' = f' + g'\]

Exemple : \((3x^2 + \sin x)' = 6x + \cos x\)

Constante multiplicative
\[(C \cdot f)' = C \cdot f'\]

Exemple : \((5x^3)' = 5 \cdot 3x^2 = 15x^2\)

Produit
\[(f \cdot g)' = f' \cdot g + f \cdot g'\]

Exemple : \((x^2 \sin x)' = 2x \sin x + x^2 \cos x\)

Quotient
\[\left(\frac{f}{g}\right)' = \frac{f'g - fg'}{g^2}\]

Exemple : \(\left(\dfrac{x^2}{x+1}\right)' = \dfrac{2x(x+1) - x^2}{(x+1)^2}\)

⚠️Pour le quotient : c'est f'g − fg' et non f'g + fg'. L'ordre et le signe moins sont critiques !
D3

Règle de la chaîne — Fonctions composées

Une fonction composée c'est une "fonction dans une autre fonction" : \(f(g(x))\).

\[\bigl(f(g(x))\bigr)' = f'(g(x)) \cdot g'(x)\]
💡En mots : dériver la fonction extérieure (en gardant l'intérieure intacte), puis multiplier par la dérivée de l'intérieure.

Cas particuliers courants (à retenir !)

FonctionDérivéeExemple
\((u)^n\) \(n \cdot u^{n-1} \cdot u'\) \((3x^2+1)^4\)' \(= 4(3x^2+1)^3 \cdot 6x\)
\(\sqrt{u}\) \(\dfrac{u'}{2\sqrt{u}}\) \((\sqrt{x^2+1})' = \dfrac{2x}{2\sqrt{x^2+1}}\)
\(e^{u}\) \(u' \cdot e^{u}\) \((e^{3x})' = 3e^{3x}\)
\(\ln(u)\) \(\dfrac{u'}{u}\) \((\ln(x^2+1))' = \dfrac{2x}{x^2+1}\)
\(\sin(u)\) \(u' \cdot \cos(u)\) \((\sin(3x))' = 3\cos(3x)\)
\(\cos(u)\) \(-u' \cdot \sin(u)\) \((\cos(5x))' = -5\sin(5x)\)
\(\tan(u)\) \(\dfrac{u'}{\cos^2(u)}\) \((\tan(2x))' = \dfrac{2}{\cos^2(2x)}\)

Exemples détaillés

\(f(x) = (x^2 - 3x + 2)^2\)

\(u = x^2 - 3x + 2 \implies u' = 2x - 3\)
\(f'(x) = 2(x^2-3x+2)^1 \cdot (2x-3) = 2(2x-3)(x^2-3x+2)\)
\(f(x) = \dfrac{3x+2}{2x-3}\) (règle du quotient)

\(f' = \dfrac{3(2x-3) - (3x+2) \cdot 2}{(2x-3)^2} = \dfrac{6x - 9 - 6x - 4}{(2x-3)^2} = \dfrac{-13}{(2x-3)^2}\)
\(f(x) = \sin(3x)\)

\(u = 3x \implies u' = 3\)
\(f'(x) = 3 \cdot \cos(3x)\)
D4

Équation de la tangente

La tangente à une courbe au point \(x = a\) est la droite qui "touche" la courbe en ce point.

\[y = f'(a) \cdot (x - a) + f(a)\]
1

Calculer \(f(a)\) — la valeur de la fonction au point donné (ordonnée du point de tangence)

2

Calculer \(f'(x)\) — la dérivée de la fonction

3

Calculer \(f'(a)\) — c'est le coefficient directeur (la pente) de la tangente

4

Substituer dans la formule \(y = f'(a)(x-a) + f(a)\)

Exemple : \(f(x) = 3x^2 - x + 1\) en \(x = 1\)

\(f(1) = 3 - 1 + 1 = 3\)
\(f'(x) = 6x - 1 \implies f'(1) = 5\)

Tangente : \(y = 5(x-1) + 3 = 5x - 5 + 3 = \mathbf{5x - 2}\)
💡La tangente en \(x=a\) donne la droite \(y = mx + b\) où \(m = f'(a)\) est la pente. Si \(f'(a) = 0\), la tangente est horizontale (maximum ou minimum local !).
D5

Optimisation — Trouver un maximum ou minimum

La dérivée permet de trouver les extrema (maximums et minimums) d'une fonction.

1

Définir la fonction à optimiser \(f(x)\) et le domaine (contraintes du problème)

2

Calculer la dérivée \(f'(x)\)

3

Résoudre \(f'(x) = 0\) pour trouver les points critiques

4

Vérifier si c'est un max ou un min : calculer \(f''(x)\) ou analyser le signe de \(f'(x)\)

5

Calculer la valeur optimale \(f(x_0)\)

Maximum local
\[f'(x_0) = 0 \text{ et } f''(x_0) < 0\]

La courbe est concave vers le bas 🔽

Minimum local
\[f'(x_0) = 0 \text{ et } f''(x_0) > 0\]

La courbe est concave vers le haut 🔼

Tableau de signe de f'(x)

\(x\)\(-\infty\)\(\cdots\)\(x_0\)\(\cdots\)\(+\infty\)
Si minimum —\(f' < 0\) (↘)\(f' = 0\)\(f' > 0\) (↗)—
Si maximum —\(f' > 0\) (↗)\(f' = 0\)\(f' < 0\) (↘)—
Exemple : Trouver le max/min de \(f(x) = x^2 - 4x + 3\)

\(f'(x) = 2x - 4\)
\(f'(x) = 0 \implies 2x - 4 = 0 \implies x = 2\)
\(f''(x) = 2 > 0 \implies\) minimum en \(x = 2\)
Valeur min : \(f(2) = 4 - 8 + 3 = -1\)

EXPONENTIELLES & LOGARITHMES
Propriétés · Calculs · Équations · Dérivées
E1

Propriétés fondamentales

Ces propriétés s'appliquent à toute base \(a > 0, a \neq 1\). Les cas les plus courants sont \(a = 10\) (log) et \(a = e\) (ln).

Puissances

\[a^1 = a\]
\[a^0 = 1\]
\[\frac{1}{a^n} = a^{-n}\]
\[a^m \cdot a^n = a^{m+n}\]
\[(a^m)^n = a^{mn}\]
\[(ab)^n = a^n b^n\]

Logarithmes \(\log_a\)

\[\log_a 1 = 0\]
\[\log_a a = 1\]
\[\log_a(xy) = \log_a x + \log_a y\]
\[\log_a\!\frac{x}{y} = \log_a x - \log_a y\]
\[\log_a x^n = n\log_a x\]
\[\log_a b = \frac{\log_c b}{\log_c a}\]

Liens exp ↔ log

\[\log_a a^x = x\]
\[a^{\log_a x} = x\]
\[\log_a 0 = \emptyset\]
\[\ln(e^x) = x\]
\[e^{\ln x} = x\]
💡Cas particuliers importants :
\(\log = \log_{10}\) (logarithme décimal)  |  \(\ln = \log_e\) (logarithme naturel/népérien)
Changement de base : \(\log_a b = \dfrac{\ln b}{\ln a} = \dfrac{\log b}{\log a}\)
⚠️\(\log(x+y) \neq \log x + \log y\) et \(\log(xy) \neq \log x \cdot \log y\) — ces erreurs sont très fréquentes !
E2

Calcul de logarithmes

Trouver la valeur exacte de \(\log_a b\)

La question est : à quelle puissance faut-il élever \(a\) pour obtenir \(b\) ?

\(\log_2 8 = ?\) → \(2^? = 8\) → \(2^3 = 8\) → \(\log_2 8 = \mathbf{3}\)
\(\log_3 \dfrac{1}{9} = ?\) → \(3^? = \dfrac{1}{9} = 3^{-2}\) → \(\log_3 \dfrac{1}{9} = \mathbf{-2}\)
\(\log_4 16 = ?\) → \(4^? = 16 = 4^2\) → \(\log_4 16 = \mathbf{2}\)

Calcul avec les propriétés (log 2 ≈ 0,3 et log 3 ≈ 0,5)

\(\log 8 = \log 2^3 = 3\log 2 \approx 3 \times 0{,}3 = \mathbf{0{,}9}\)
\(\log 12 = \log(4 \times 3) = \log 4 + \log 3 = 2\log 2 + \log 3 \approx 0{,}6 + 0{,}5 = \mathbf{1{,}1}\)
\(\log 0{,}2 = \log\!\dfrac{2}{10} = \log 2 - \log 10 = \log 2 - 1 \approx 0{,}3 - 1 = \mathbf{-0{,}7}\)
\(\log 5 = \log\!\dfrac{10}{2} = \log 10 - \log 2 = 1 - 0{,}3 = \mathbf{0{,}7}\)
E3

Équations logarithmiques

Pour résoudre une équation avec des logarithmes, l'objectif est de regrouper tous les log d'un côté en un seul logarithme, puis d'exponentier.

1

Utiliser les propriétés pour regrouper les log : \(\log a + \log b = \log(ab)\), \(\log a - \log b = \log\frac{a}{b}\), \(n\log a = \log a^n\)

2

Isoler le logarithme : \(\log_a(\ldots) = c\)

3

Exponentier : si \(\log_a f(x) = c\) alors \(f(x) = a^c\)

4

Vérifier que les arguments des log sont strictement positifs (le log d'un nombre ≤ 0 n'existe pas !)

Exemples

\(\log x + \log 3 = \log 7\)

\(\log(3x) = \log 7\)
\(3x = 7 \implies x = \dfrac{7}{3}\) ✓ (positif)
\(\log 2x - \log 4 = 2\log 5\)

\(\log\!\dfrac{2x}{4} = \log 25\)
\(\dfrac{2x}{4} = 25 \implies 2x = 100 \implies x = 50\) ✓
\(\ln 3x + \ln 5 = 2 - \ln 3\)

\(\ln(15x) = 2 - \ln 3\)
\(\ln(15x) + \ln 3 = 2\)
\(\ln(45x) = 2 \implies 45x = e^2 \implies x = \dfrac{e^2}{45}\)
\(\log x = \dfrac{3}{2}\)

\(x = 10^{3/2} = 10\sqrt{10} \approx 31{,}6\)
⚠️Toujours vérifier à la fin que les arguments de tous les log sont strictement positifs. Une solution négative ou nulle est à rejeter !
E4

Équations exponentielles

Pour résoudre une équation avec des exponentielles, on prend le log ou ln des deux membres, ou on ramène à une même base.

1

Si possible, ramener à la même base : \(a^{f(x)} = a^{g(x)} \implies f(x) = g(x)\)

2

Sinon, appliquer ln (ou log) des deux côtés

3

Utiliser \(\ln(a^x) = x \ln a\) pour "descendre" l'exposant

4

Résoudre l'équation obtenue (souvent linéaire ou quadratique)

Exemples

\(3^x = 5\)

\(\ln(3^x) = \ln 5\)
\(x \ln 3 = \ln 5\)
\(x = \dfrac{\ln 5}{\ln 3}\)
\(2^{2x} + 2^x - 6 = 0\) (équation du 2e degré en \(2^x\))

Poser \(u = 2^x\) :\quad \(u^2 + u - 6 = 0\)
\((u+3)(u-2) = 0 \implies u = 2\) (car \(u = 2^x > 0\), donc \(u = -3\) rejeté)
\(2^x = 2 \implies x = 1\)
\(5^{x^2+4x+3} = 125\)

\(5^{x^2+4x+3} = 5^3\)
\(x^2 + 4x + 3 = 3\)
\(x^2 + 4x = 0 \implies x(x+4) = 0\)
\(x = 0\) ou \(x = -4\)
\(3 \cdot 2^x - 5 = 0\)

\(2^x = \dfrac{5}{3}\)
\(x = \log_2\!\dfrac{5}{3} = \dfrac{\ln(5/3)}{\ln 2}\)
💡Si tu vois \(e^{2x} + e^x - \ldots\), pose \(u = e^x\) → tu obtiens une équation du 2e degré en \(u\) !
E5

Dérivées des fonctions exponentielles et logarithmiques

Fonction \(f(x)\)Dérivée \(f'(x)\)Exemple
\(e^x\) \(e^x\) \((e^x)' = e^x\)
\(e^{u(x)}\) \(u'(x) \cdot e^{u(x)}\) \((e^{3x})' = 3e^{3x}\)
\(a^x\) \(a^x \ln a\) \((5^x)' = 5^x \ln 5\)
\(a^{u(x)}\) \(u'(x) \cdot a^{u(x)} \ln a\) \((2^{\sin x})' = \cos x \cdot 2^{\sin x} \ln 2\)
\(\ln x\) \(\dfrac{1}{x}\) \((\ln x)' = \dfrac{1}{x}\)
\(\ln(u(x))\) \(\dfrac{u'(x)}{u(x)}\) \((\ln(x^2+1))' = \dfrac{2x}{x^2+1}\)
\(\log_a x\) \(\dfrac{1}{x \ln a}\) \((\log_2 x)' = \dfrac{1}{x \ln 2}\)
\(\ln x^2 = 2\ln x\) \(\dfrac{2}{x}\) \((\ln x^2)' = \dfrac{2}{x}\)

Exemples détaillés avec règle de la chaîne

\(f(x) = e^x(\sin x + \cos x)\) (produit)

\(f' = e^x(\sin x + \cos x) + e^x(\cos x - \sin x)\)
\(= e^x(\sin x + \cos x + \cos x - \sin x) = \mathbf{2e^x \cos x}\)
\(f(x) = \ln(\sin x)\) (composée)

\(u = \sin x \implies u' = \cos x\)
\(f'(x) = \dfrac{\cos x}{\sin x} = \cot x\)
\(f(x) = x^2 \ln x\) (produit)

\(f' = 2x \cdot \ln x + x^2 \cdot \dfrac{1}{x} = 2x\ln x + x = x(2\ln x + 1)\)
\(f(x) = 2^{\sin^2 x}\) (double composée)

\(u = \sin^2 x \implies u' = 2\sin x \cos x = \sin(2x)\)
\(f'(x) = \sin(2x) \cdot 2^{\sin^2 x} \cdot \ln 2\)

⚠️\(\log(x+y) \neq \log x + \log y\) et \(\log(xy) \neq \log x \cdot \log y\) — ces erreurs sont très fréquentes !
E2

Calcul de logarithmes

Trouver la valeur exacte de \(\log_a b\)

La question est : à quelle puissance faut-il élever \(a\) pour obtenir \(b\) ?

\(\log_2 8 = ?\) → \(2^? = 8\) → \(2^3 = 8\) → \(\log_2 8 = \mathbf{3}\)
\(\log_3 \dfrac{1}{9} = ?\) → \(3^? = \dfrac{1}{9} = 3^{-2}\) → \(\log_3 \dfrac{1}{9} = \mathbf{-2}\)
\(\log_4 16 = ?\) → \(4^? = 16 = 4^2\) → \(\log_4 16 = \mathbf{2}\)

Calcul avec les propriétés (log 2 ≈ 0,3 et log 3 ≈ 0,5)

\(\log 8 = \log 2^3 = 3\log 2 \approx 3 \times 0{,}3 = \mathbf{0{,}9}\)
\(\log 12 = \log(4 \times 3) = \log 4 + \log 3 = 2\log 2 + \log 3 \approx 0{,}6 + 0{,}5 = \mathbf{1{,}1}\)
\(\log 0{,}2 = \log\!\dfrac{2}{10} = \log 2 - \log 10 = \log 2 - 1 \approx 0{,}3 - 1 = \mathbf{-0{,}7}\)
\(\log 5 = \log\!\dfrac{10}{2} = \log 10 - \log 2 = 1 - 0{,}3 = \mathbf{0{,}7}\)
E3

Équations logarithmiques

Pour résoudre une équation avec des logarithmes, l'objectif est de regrouper tous les log d'un côté en un seul logarithme, puis d'exponentier.

1

Utiliser les propriétés pour regrouper les log : \(\log a + \log b = \log(ab)\), \(\log a - \log b = \log\frac{a}{b}\), \(n\log a = \log a^n\)

2

Isoler le logarithme : \(\log_a(\ldots) = c\)

3

Exponentier : si \(\log_a f(x) = c\) alors \(f(x) = a^c\)

4

Vérifier que les arguments des log sont strictement positifs (le log d'un nombre ≤ 0 n'existe pas !)

Exemples

\(\log x + \log 3 = \log 7\)

\(\log(3x) = \log 7\)
\(3x = 7 \implies x = \dfrac{7}{3}\) ✓ (positif)
\(\log 2x - \log 4 = 2\log 5\)

\(\log\!\dfrac{2x}{4} = \log 25\)
\(\dfrac{2x}{4} = 25 \implies 2x = 100 \implies x = 50\) ✓
\(\ln 3x + \ln 5 = 2 - \ln 3\)

\(\ln(15x) = 2 - \ln 3\)
\(\ln(15x) + \ln 3 = 2\)
\(\ln(45x) = 2 \implies 45x = e^2 \implies x = \dfrac{e^2}{45}\)
\(\log x = \dfrac{3}{2}\)

\(x = 10^{3/2} = 10\sqrt{10} \approx 31{,}6\)
⚠️Toujours vérifier à la fin que les arguments de tous les log sont strictement positifs. Une solution négative ou nulle est à rejeter !
E4

Équations exponentielles

Pour résoudre une équation avec des exponentielles, on prend le log ou ln des deux membres, ou on ramène à une même base.

1

Si possible, ramener à la même base : \(a^{f(x)} = a^{g(x)} \implies f(x) = g(x)\)

2

Sinon, appliquer ln (ou log) des deux côtés

3

Utiliser \(\ln(a^x) = x \ln a\) pour "descendre" l'exposant

4

Résoudre l'équation obtenue (souvent linéaire ou quadratique)

Exemples

\(3^x = 5\)

\(\ln(3^x) = \ln 5\)
\(x \ln 3 = \ln 5\)
\(x = \dfrac{\ln 5}{\ln 3}\)
\(2^{2x} + 2^x - 6 = 0\) (équation du 2e degré en \(2^x\))

Poser \(u = 2^x\) :\quad \(u^2 + u - 6 = 0\)
\((u+3)(u-2) = 0 \implies u = 2\) (car \(u = 2^x > 0\), donc \(u = -3\) rejeté)
\(2^x = 2 \implies x = 1\)
\(5^{x^2+4x+3} = 125\)

\(5^{x^2+4x+3} = 5^3\)
\(x^2 + 4x + 3 = 3\)
\(x^2 + 4x = 0 \implies x(x+4) = 0\)
\(x = 0\) ou \(x = -4\)
\(3 \cdot 2^x - 5 = 0\)

\(2^x = \dfrac{5}{3}\)
\(x = \log_2\!\dfrac{5}{3} = \dfrac{\ln(5/3)}{\ln 2}\)
💡Si tu vois \(e^{2x} + e^x - \ldots\), pose \(u = e^x\) → tu obtiens une équation du 2e degré en \(u\) !
E5

Dérivées des fonctions exponentielles et logarithmiques

Fonction \(f(x)\)Dérivée \(f'(x)\)Exemple
\(e^x\) \(e^x\) \((e^x)' = e^x\)
\(e^{u(x)}\) \(u'(x) \cdot e^{u(x)}\) \((e^{3x})' = 3e^{3x}\)
\(a^x\) \(a^x \ln a\) \((5^x)' = 5^x \ln 5\)
\(a^{u(x)}\) \(u'(x) \cdot a^{u(x)} \ln a\) \((2^{\sin x})' = \cos x \cdot 2^{\sin x} \ln 2\)
\(\ln x\) \(\dfrac{1}{x}\) \((\ln x)' = \dfrac{1}{x}\)
\(\ln(u(x))\) \(\dfrac{u'(x)}{u(x)}\) \((\ln(x^2+1))' = \dfrac{2x}{x^2+1}\)
\(\log_a x\) \(\dfrac{1}{x \ln a}\) \((\log_2 x)' = \dfrac{1}{x \ln 2}\)
\(\ln x^2 = 2\ln x\) \(\dfrac{2}{x}\) \((\ln x^2)' = \dfrac{2}{x}\)

Exemples détaillés avec règle de la chaîne

\(f(x) = e^x(\sin x + \cos x)\) (produit)

\(f' = e^x(\sin x + \cos x) + e^x(\cos x - \sin x)\)
\(= e^x(\sin x + \cos x + \cos x - \sin x) = \mathbf{2e^x \cos x}\)
\(f(x) = \ln(\sin x)\) (composée)

\(u = \sin x \implies u' = \cos x\)
\(f'(x) = \dfrac{\cos x}{\sin x} = \cot x\)
\(f(x) = x^2 \ln x\) (produit)

\(f' = 2x \cdot \ln x + x^2 \cdot \dfrac{1}{x} = 2x\ln x + x = x(2\ln x + 1)\)
\(f(x) = 2^{\sin^2 x}\) (double composée)

\(u = \sin^2 x \implies u' = 2\sin x \cos x = \sin(2x)\)
\(f'(x) = \sin(2x) \cdot 2^{\sin^2 x} \cdot \ln 2\)
