Tableau fondamental ⚠ À connaître par cœur !

Angle \(0\) \(\dfrac{\pi}{6}\ (30°)\) \(\dfrac{\pi}{4}\ (45°)\) \(\dfrac{\pi}{3}\ (60°)\) \(\dfrac{\pi}{2}\ (90°)\)
\(\cos\) \(1\) \(\dfrac{\sqrt{3}}{2}\) \(\dfrac{\sqrt{2}}{2}\) \(\dfrac{1}{2}\) \(0\)
\(\sin\) \(0\) \(\dfrac{1}{2}\) \(\dfrac{\sqrt{2}}{2}\) \(\dfrac{\sqrt{3}}{2}\) \(1\)
\(\tan\) \(0\) \(\dfrac{\sqrt{3}}{3}\) \(1\) \(\sqrt{3}\) \(\nexists\)
💡Astuce pour sin : pense à \(\frac{\sqrt{0}}{2},\ \frac{\sqrt{1}}{2},\ \frac{\sqrt{2}}{2},\ \frac{\sqrt{3}}{2},\ \frac{\sqrt{4}}{2}\) pour 0°, 30°, 45°, 60°, 90°.
💡Astuce pour cos : c'est l'ordre inverse ! \(\frac{\sqrt{4}}{2},\ \frac{\sqrt{3}}{2},\ \frac{\sqrt{2}}{2},\ \frac{\sqrt{1}}{2},\ \frac{\sqrt{0}}{2}\)
2

Signes par quadrant

Le cercle est divisé en 4 quadrants. Les signes de \(\cos\), \(\sin\) et \(\tan\) changent selon le quadrant.

1er quadrant — 0° à 90°
\(\cos > 0 \quad \sin > 0 \quad \tan > 0\)

→ Tout est positif ✓

2ème quadrant — 90° à 180°
\(\cos < 0 \quad \sin > 0 \quad \tan < 0\)

→ Seul sin est positif

3ème quadrant — 180° à 270°
\(\cos < 0 \quad \sin < 0 \quad \tan > 0\)

→ Seul tan est positif

4ème quadrant — 270° à 360°
\(\cos > 0 \quad \sin < 0 \quad \tan < 0\)

→ Seul cos est positif

💡Moyen mnémotechnique : Tout — Sin — Tan — Cos → dans chaque quadrant (1→2→3→4), c'est ce qui est positif.
3

Angles négatifs & grands angles

Règles pour les angles négatifs

Cosinus
\[\cos(-x) = \cos(x)\]

✓ Le signe ne change pas

Sinus
\[\sin(-x) = -\sin(x)\]

✗ Le signe change !

Tangente
\[\tan(-x) = -\tan(x)\]

✗ Le signe change !

⚠️Seul \(\cos\) garde son signe quand l'angle est négatif. Sin et tan changent de signe !

Grands angles — ramener entre 0 et 2π

Si l'angle est trop grand (> 2π) : soustraire 2π autant de fois que nécessaire.

Si l'angle est négatif : ajouter 2π autant de fois que nécessaire.

Exemple : \(\quad -\dfrac{\pi}{2} = -90° \xrightarrow{+360°} 270° = \dfrac{3\pi}{2}\)
Exemple : \(\quad \dfrac{7\pi}{3} \xrightarrow{-2\pi} \dfrac{7\pi}{3} - \dfrac{6\pi}{3} = \dfrac{\pi}{3}\)

Conversion degrés ↔ radians

\[\text{rad} = \text{deg} \times \frac{\pi}{180} \qquad\qquad \text{deg} = \text{rad} \times \frac{180}{\pi}\]
4

Résoudre des équations trigonométriques

💡Il y a toujours une infinité de solutions. Le \(k\) représente n'importe quel entier (\(0, \pm1, \pm2, \ldots\))

Équation avec sin

\[\sin(x) = k \implies x = \arcsin(k) + 2k\pi \quad \text{ou} \quad x = \pi - \arcsin(k) + 2k\pi\]

Exemple : \(\sin(x) = \dfrac{1}{2}\) → on reconnaît \(\arcsin\!\left(\dfrac{1}{2}\right) = \dfrac{\pi}{6}\), donc :

\[x = \frac{\pi}{6} + 2k\pi \qquad \text{ou} \qquad x = \pi - \frac{\pi}{6} + 2k\pi = \frac{5\pi}{6} + 2k\pi\]

Équation avec cos

\[\cos(x) = k \implies x = \arccos(k) + 2k\pi \quad \text{ou} \quad x = -\arccos(k) + 2k\pi\]

Exemple : \(\cos(x) = \dfrac{\sqrt{2}}{2}\) → on reconnaît \(\arccos\!\left(\dfrac{\sqrt{2}}{2}\right) = \dfrac{\pi}{4}\), donc :

\[x = \frac{\pi}{4} + 2k\pi \qquad \text{ou} \qquad x = -\frac{\pi}{4} + 2k\pi\]

Équation avec tan

\[\tan(x) = k \implies x = \arctan(k) + k\pi \quad \text{(une seule famille de solutions)}\]
💡Pour \(\tan\) : on ajoute \(k\pi\) et non \(2k\pi\), car la tangente se répète tous les 180°.

Cas avec sin(2x), cos(3x)…

Méthode : poser \(u = 2x\), résoudre en \(u\), puis diviser par 2 à la fin.

Exemple : \(\sin(2x) = \dfrac{\sqrt{3}}{2}\) → \(\arcsin\!\left(\dfrac{\sqrt{3}}{2}\right) = \dfrac{\pi}{3}\)

\[2x = \frac{\pi}{3} + 2k\pi \implies x = \frac{\pi}{6} + k\pi\] \[2x = \frac{2\pi}{3} + 2k\pi \implies x = \frac{\pi}{3} + k\pi\]
5

Identités trigonométriques

Identité fondamentale (la plus importante !)

\[\cos^2(x) + \sin^2(x) = 1\]
On en tire aussi
\[\cos^2(x) = 1 - \sin^2(x)\]
Et aussi
\[\sin^2(x) = 1 - \cos^2(x)\]

Formules du double angle

\[\cos(2x) = \cos^2(x) - \sin^2(x) = 2\cos^2(x) - 1 = 1 - 2\sin^2(x)\]
\[\sin(2x) = 2\sin(x)\cos(x)\]
\[\tan(2x) = \frac{2\tan(x)}{1 - \tan^2(x)}\]

Formules d'addition

\[\cos(a+b) = \cos a\cos b - \sin a\sin b\]
\[\cos(a-b) = \cos a\cos b + \sin a\sin b\]
\[\sin(a+b) = \sin a\cos b + \cos a\sin b\]
\[\sin(a-b) = \sin a\cos b - \cos a\sin b\]

Formules somme → produit

\[\sin p + \sin q = 2\sin\!\left(\frac{p+q}{2}\right)\cos\!\left(\frac{p-q}{2}\right)\]
\[\cos p + \cos q = 2\cos\!\left(\frac{p+q}{2}\right)\cos\!\left(\frac{p-q}{2}\right)\]
\[\cos p - \cos q = -2\sin\!\left(\frac{p+q}{2}\right)\sin\!\left(\frac{p-q}{2}\right)\]
6

Périodicité — Lire & écrire une fonction

Une fonction sinusoïdale s'écrit :

\[f(t) = A \cdot \sin(\omega t + \varphi)\]
ParamètreNomComment le trouver sur le graphe
\(A\) Amplitude \(A = \dfrac{y_{max} - y_{min}}{2}\)
\(T\) Période Distance entre 2 maximums consécutifs
\(\omega\) Pulsation \(\omega = \dfrac{2\pi}{T}\)
\(\varphi\) Déphasage Valeur de \(t\) quand la fonction atteint son max (pour sin)
💡Pour convertir un cosinus en sinus : \(\quad \cos(x) = \sin\!\left(x + \dfrac{\pi}{2}\right)\)
