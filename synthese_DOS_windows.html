// Répertoires

Batch & Variables
Fichiers .bat — automatiser des séquences de commandes DOS
📄 DéfinitionUn fichier .bat (ou .cmd) est un fichier texte contenant des commandes DOS exécutées automatiquement dans l'ordre.
// Instructions de base
@ECHO OFFMasquer les commandes
Mettre en première ligne pour ne pas afficher chaque commande
@echo off REM les commandes s'exécutent sans s'afficher
REMCommentaires
Lignes non exécutées, pour documenter
REM Auteur : Liam REM Date : 2025
ECHOAfficher un message
echo. pour une ligne vide
echo Bonjour ! echo. ← ligne vide
PAUSEAttendre une touche
Suspend l'exécution
pause
COLORCouleur de la console
2 chiffres hex — fond puis texte
color F0 ← fond blanc, texte noir color 0A ← fond noir, texte vert
PROMPTInvite de commande
$D=date, $P=chemin, $G= >
prompt $D $P$G
// Variables d'environnement
REM Afficher toutes les variables : set REM Définir une variable : set MONVAR=bonjour echo %MONVAR% REM Variables prédéfinies : echo %USERPROFILE% ← C:\Users\Liam echo %USERNAME% ← Liam echo %COMPUTERNAME% ← nom du PC echo %PATH% ← chemins des exécutables echo %TEMP% ← dossier temporaire
⚠ PATH dans un batchLa modification du PATH dans un batch n'est valide que pour la console en cours. Pour une modification permanente : Propriétés système → Variables d'environnement.
set PATH=%PATH%;%HOMEDRIVE%%HOMEPATH%\Mes documents\Mes Batch

Partages réseau, lecteurs réseau, batch automatique au démarrage

connecter.bat

deconnecter.bat
