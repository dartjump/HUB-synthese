P1

Équation de droite — 1er degré

\[y = ax + b\]

où \(a\) est le coefficient angulaire (pente) et \(b\) est l'ordonnée à l'origine.

Pente entre deux points
\[a = \frac{y_2 - y_1}{x_2 - x_1}\]
Trouver b

Substituer un point connu \((x_0, y_0)\) dans \(y = ax + b\) et résoudre pour \(b\).

Cas particuliers

Droites parallèles

Même pente \(a\), ordonnée \(b\) différente.

\(y=2x+3\) ∥ \(y=2x-6\)
Droites perpendiculaires
\[a_1 \times a_2 = -1\]

Les pentes sont inverses et opposées.

Pente en %

100% de pente = coefficient angulaire de 1.
Donc \(y = 1\cdot x + b\).

💡Pour une droite passant par l'axe OX en \(x_0\) et l'axe OY en \(y_0\) :
Deux points connus : \((x_0, 0)\) et \((0, y_0)\) → calculer la pente, puis \(b = y_0\).
(a) Passe par (0,0) et (4,−1) : \(a=\frac{-1}{4}\), \(b=0\) → \(d_1 : y = -\frac{1}{4}x\)
(b) Point (2,2), pente \(\frac{1}{3}\) : \(2=\frac{1}{3}\cdot 2+b\) → \(b=\frac{4}{3}\) → \(d_2 : y = \frac{x+4}{3}\)
(c) Point (1,2), pente 100% = 1 : \(2=1\cdot 1+b\) → \(b=1\) → \(d_3 : y = x+1\)
(d) Points (1,2) et (−2,−1) : \(a=\frac{-1-2}{-2-1}=1\), \(b=1\) → \(d_4 : y = x+1\)
(e) Points (0,1) et (1,3) : \(a=2\), \(b=1\) → \(d_5 : y = 2x+1\)
(f) Passe par l'origine, \(a=1{,}5=\frac{3}{2}\) → \(d_6 : y = \frac{3}{2}x\)
(g) Parallèle à \(y=2x+3\) → même pente \(a=2\). Point (2,−2) : \(-2=4+b\) → \(b=-6\) → \(d_7 : y = 2x-6\)
(h) Perpendiculaire à \(y=-x+\frac{1}{2}\) (pente −1) → pente = 1. Point (−1,0) : \(0=−1+b\) → \(b=1\) → \(d_8 : y = x+1\)
(i) Coupe OX en 5 (point (5,0)) et OY en −2 (point (0,−2)) : \(a=\frac{0-(-2)}{5-0}=\frac{2}{5}\), \(b=-2\) → \(d_9 : y = \frac{2}{5}x-2\) (a) Point (4,2) : \(2 = 4-2 = 2\) ✓ → Oui, le point appartient à la droite.
(b) Point (4,1) : \(1 \neq 4-2 = 2\) → Non, le point n'appartient pas à la droite.
P2

Distance — entre deux points et point-droite

Distance entre deux points

\[d = \sqrt{(x_2-x_1)^2 + (y_2-y_1)^2}\]

Distance d'un point \((x_0, y_0)\) à une droite \(ax + by + c = 0\)

\[d = \frac{|ax_0 + by_0 + c|}{\sqrt{a^2+b^2}}\]
💡Pour utiliser la formule point-droite, il faut d'abord mettre la droite sous la forme \(ax+by+c=0\).
Exemple : \(y = 2x-2\) → \(-2x+y+2=0\) → \(a=-2,\ b=1,\ c=2\)
1

Réécrire la droite \(y=mx+p\) sous la forme \(-mx+y-p=0\)

2

Identifier \(a=-m,\ b=1,\ c=-p\)

3

Appliquer \(d = \dfrac{|ax_0+by_0+c|}{\sqrt{a^2+b^2}}\)

(a) (0,0) et (3,2) : \(d=\sqrt{9+4}=\sqrt{13}\)
(b) (−1,3) et (2,6) : \(d=\sqrt{9+9}=3\sqrt{2}\)
(c) (2,−1) et (2,−3) : \(d=\sqrt{0+4}=2\)
(d) (0,6) et (3,2) : \(d=\sqrt{9+16}=5\)
(e) \(d_1: y=2x-2\) et (4,1) → \(-2x+y+2=0\) : \(d=\dfrac{|-8+1+2|}{\sqrt{5}}=\dfrac{5}{\sqrt{5}}=\sqrt{5}\)
(f) \(d_2: y=x\) et (3,3) → \(-x+y=0\) : \(d=\dfrac{|-3+3|}{\sqrt{2}}=0\) → le point est sur la droite
(g) \(d_3: y=2x\) et (1,−2) → \(-2x+y=0\) : \(d=\dfrac{|-2-2|}{\sqrt{5}}=\dfrac{4}{\sqrt{5}}=\dfrac{2}{5}\sqrt{20}\)
P3

Intersection de droites

Pour trouver l'intersection de \(d_1\) et \(d_2\) : résoudre le système en égalisant les deux expressions de \(y\).

1

Écrire les deux équations sous la forme \(y = \ldots\)

2

Égaliser : \(a_1x+b_1 = a_2x+b_2\) → résoudre en \(x\)

3

Substituer \(x\) dans l'une des équations pour trouver \(y\)

4

Si les droites sont parallèles (même pente) → aucune intersection. Si confondues → infinité de solutions.

\(d_1: y=2x-2\) et \(d_2: 2y=x-6\) → \(y=\frac{x-6}{2}\)
\(2x-2 = \frac{x-6}{2}\) → \(4x-4=x-6\) → \(3x=-2\) → \(x=-\frac{2}{3}\)
\(y=2\cdot(-\frac{2}{3})-2=-\frac{4}{3}-2=-\frac{10}{3}\) → intersection : \(\left(-\frac{2}{3},\ -\frac{10}{3}\right)\)

\(d_1: y=2x-2\) et \(d_3: 3y=6x-6\) → \(y=2x-2\)
Les deux droites ont la même équation → elles sont confondues (toute la droite est solution).

Problèmes du 1er degré

💡Méthode : nommer l'inconnue \(x\), traduire les conditions en équation, résoudre, vérifier.
Soit la benjamine = \(x\), cadette = \(x+10\), aînée = \(x+30\)
\(x + (x+10) + (x+30) = 100\) → \(3x+40=100\) → \(x=20\)
→ Benjamine : 20€, cadette : 30€, aînée : 50€ Soit les trois entiers : \(n-1,\ n,\ n+1\)
\((n-1)+n+(n+1)=2010\) → \(3n=2010\) → \(n=670\)
→ Les nombres sont 669, 670, 671 \(\dfrac{x+45}{x+59} = \dfrac{7}{9}\)
\(9(x+45) = 7(x+59)\) → \(9x+405=7x+413\) → \(2x=8\) → \(x=4\)

Équations et fonctions du second degré

\[f(x) = ax^2 + bx + c \qquad (a \neq 0)\]
Discriminant
\[\Delta = b^2 - 4ac\]

\(\Delta > 0\) : 2 racines réelles
\(\Delta = 0\) : 1 racine double
\(\Delta < 0\) : pas de racines réelles

Racines
\[x_{1,2} = \frac{-b \pm \sqrt{\Delta}}{2a}\]
Sommet (vertex)
\[x_S = \frac{-b}{2a}\]
\[y_S = f(x_S)\]
Concavité

• \(a > 0\) : parabole vers le haut ∪ (minimum au sommet)
• \(a < 0\) : parabole vers le bas ∩ (maximum au sommet)

Forme factorisée
\[f(x) = a(x - x_1)(x - x_2)\]

si \(x_1\) et \(x_2\) sont les racines connues.

💡Lien somme/produit des racines :
\(x_1 + x_2 = -\dfrac{b}{a}\)  ·  \(x_1 \cdot x_2 = \dfrac{c}{a}\)
Utile pour retrouver une équation dont on connaît les racines.
P6

Méthode — étudier et dessiner une parabole

1

Identifier \(a, b, c\) et déterminer la concavité (\(a>0\) : ∪ ; \(a<0\) : ∩)

2

Calculer \(\Delta = b^2-4ac\) → déterminer le nombre de racines

3

Calculer les racines \(x_{1,2} = \frac{-b\pm\sqrt{\Delta}}{2a}\)

4

Calculer le sommet : \(x_S = \frac{-b}{2a}\), puis \(y_S = f(x_S)\)

5

Calculer l'ordonnée à l'origine : \(f(0) = c\)

6

Tracer : placer les racines, le sommet, l'ordonnée à l'origine

(a) \(f_1(x) = (2x+2)(2x-2) = 4x^2-4\)
\(a=4>0\) → ∪  ·  Racines : \(2x+2=0\Rightarrow x=-1\) et \(2x-2=0\Rightarrow x=1\)
Sommet : \(x_S=0\), \(y_S=-4\) → Sommet \((0,-4)\)

(b) \(f_2(x) = -x(x-1) = -x^2+x\)
\(a=-1<0\) → ∩  ·  Racines : \(x=0\) et \(x=1\)
Sommet : \(x_S=\frac{1}{2}\), \(y_S=\frac{1}{4}\) → Maximum \(\left(\frac{1}{2},\frac{1}{4}\right)\) (car \(a<0\))

(c) \(f_3(x) = x^2-x-2\)
\(a=1>0\) → ∪  ·  \(\Delta=1+8=9\) → \(x_{1,2}=\frac{1\pm 3}{2}\) → \(x_1=-1,\ x_2=2\)
Sommet : \(x_S=\frac{1}{2}\), \(y_S=\frac{1}{4}-\frac{1}{2}-2=-\frac{9}{4}\) → Sommet \(\left(\frac{1}{2},-\frac{9}{4}\right)\)

(d) \(f_4(x) = x^2-x-12\)
\(a=1>0\) → ∪  ·  \(\Delta=1+48=49\) → \(x_{1,2}=\frac{1\pm 7}{2}\) → \(x_1=-3,\ x_2=4\)
Sommet : \(x_S=\frac{1}{2}\), \(y_S=-\frac{49}{4}\) → Sommet \(\left(\frac{1}{2},-\frac{49}{4}\right)\) (a) \(x + \frac{1}{x} = 2\) → multiplier par \(x\) : \(x^2+1=2x\) → \(x^2-2x+1=0\) → \((x-1)^2=0\) → \(x=1\)

(b) \(\frac{2x-3}{x-1} = x-1\) → multiplier par \((x-1)\) : \(2x-3=(x-1)^2=x^2-2x+1\)
→ \(x^2-4x+4=0\) → \((x-2)^2=0\) → \(x=2\) Racines connues : \(x_1=3\) et \(x_2=-4\)
\(f(x) = a(x-3)(x+4),\quad a\in\mathbb{R}\) Méthode : si somme = \(S\) et produit = \(P\), les deux nombres sont racines de \(x^2 - Sx + P = 0\).

(a) \(S=4,\ P=3\) → \(x^2-4x+3=0\) → \(\{1,\ 3\}\)
(b) \(S=7,\ P=10\) → \(x^2-7x+10=0\) → \(\{2,\ 5\}\)
(c) \(S=1,\ P=-2\) → \(x^2-x-2=0\) → \(\{-1,\ 2\}\)
(d) \(S=-5,\ P=6\) → \(x^2+5x+6=0\) → \(\{-2,\ -3\}\)
(e) \(S=\frac{3}{2},\ P=\frac{1}{2}\) → \(x^2-\frac{3}{2}x+\frac{1}{2}=0\) → \(\left\{\frac{1}{2},\ 1\right\}\) Soit \(n-1,\ n,\ n+1\) : \((n-1)^2+n^2+(n+1)^2=50\)
\(n^2-2n+1+n^2+n^2+2n+1=50\) → \(3n^2+2=50\) → \(n^2=16\) → \(n=\pm 4\)
→ Solutions : \(\{3,4,5\}\) ou \(\{-5,-4,-3\}\)
P7

Intersection droite / parabole

1

Égaliser : \(f(x) = d(x)\) → ramener à \(ax^2+bx+c=0\)

2

Calculer \(\Delta\) → \(\Delta>0\) : 2 intersections ; \(\Delta=0\) : tangente ; \(\Delta<0\) : aucune

3

Calculer les \(x\) d'intersection, puis les \(y\) correspondants

\(x^2-x-2 = x+1\) → \(x^2-2x-3=0\)
\(\Delta=4+12=16\) → \(x_{1,2}=\frac{2\pm 4}{2}\) → \(x_1=3\) et \(x_2=-1\)
\(y_1=3+1=4\) et \(y_2=-1+1=0\)
→ Intersections : \(\mathbf{(3,4)}\) et \(\mathbf{(-1,0)}\)

\(y=2(2)-1=3\) → \(3=3\) ✓ → Oui

c) Distance entre la droite \(y=2x-1\) et le point (2,−2) :

\(\Delta=4+12=16\) → \(x_{1,2}=\frac{-2\pm 4}{-2}\) → \(x_1=-1,\ x_2=3\)
Sommet : \(x_S=\frac{-2}{-2}=1\), \(y_S=-1+2+3=4\) → Sommet \((1,4)\)

\(\Delta=9+4=13\) → \(x_{1,2}=\frac{3\pm\sqrt{13}}{2}\)
\(x_1=\frac{3+\sqrt{13}}{2}\approx 3{,}30\) → \(y_1\approx 8{,}90\)
\(x_2=\frac{3-\sqrt{13}}{2}\approx -0{,}30\) → \(y_2\approx -1{,}90\) Soit \(p\) = prix d'une balle. Raquette = \(p+16\)
