C1

Les trois représentations d'un nombre complexe

Un nombre complexe peut s'écrire de trois façons différentes, toutes équivalentes.

Cartésienne (Algébrique)
\(z = a + jb\)
\(a = \text{Re}(z)\) : partie réelle
\(b = \text{Im}(z)\) : partie imaginaire
Idéale pour additionner et soustraire
Polaire (Trigonométrique)
\(z = r\,\text{cjs}(\theta)\)
\(r = |z|\) : module
\(\theta\) : argument (en rad ou degrés)
Idéale pour multiplier et diviser
Exponentielle
\(z = re^{j\theta}\)
\(r = |z|\) : module
\(\theta\) : argument
Identique à la polaire, notation compacte
💡Note : en électronique/maths belges on utilise \(j\) à la place de \(i\) pour l'unité imaginaire. C'est la même chose : \(j^2 = -1\)
C2

Conversions entre les formes

Cartésienne → Polaire/Exponentielle

On cherche le module \(r\) et l'argument \(\theta\) à partir de \(a\) et \(b\).

Module
\[r = |z| = \sqrt{a^2 + b^2}\]
Argument
\[\theta = \arctan\!\left(\frac{b}{a}\right)\]

⚠ Attention au quadrant ! (voir ci-dessous)

⚠️L'argument dépend du quadrant ! La calculatrice donne arctan dans \([-90°, 90°]\), mais \(\theta\) peut être dans n'importe quel quadrant selon les signes de \(a\) et \(b\) :
• \(a > 0\) : \(\theta = \arctan(b/a)\)
• \(a < 0, b \geq 0\) : \(\theta = \arctan(b/a) + 180°\)
• \(a < 0, b < 0\) : \(\theta = \arctan(b/a) - 180°\)
• \(a = 0, b > 0\) : \(\theta = 90°\)  |  \(a = 0, b < 0\) : \(\theta = -90°\)
Exemple : \(z = 1 - j\) → \(a = 1,\ b = -1\)

\(r = \sqrt{1^2 + (-1)^2} = \sqrt{2}\)
\(\theta = \arctan\!\left(\dfrac{-1}{1}\right) = -45° = -\dfrac{\pi}{4}\) (4e quadrant, \(a > 0\) ✓)

\(\Rightarrow z = \sqrt{2}\,\text{cjs}(-45°) = \sqrt{2}\,e^{-j\pi/4}\)
Exemple : \(z = -\sqrt{3} + j\) → \(a = -\sqrt{3},\ b = 1\)

\(r = \sqrt{3 + 1} = 2\)
\(\theta = \arctan\!\left(\dfrac{1}{-\sqrt{3}}\right) + 180° = -30° + 180° = 150° = \dfrac{5\pi}{6}\)

\(\Rightarrow z = 2\,\text{cjs}(150°)\)

Polaire/Exponentielle → Cartésienne

On cherche \(a\) et \(b\) à partir de \(r\) et \(\theta\).

Partie réelle
\[a = r\cos(\theta)\]
Partie imaginaire
\[b = r\sin(\theta)\]
Exemple : \(z = 2\,\text{cjs}\!\left(\dfrac{\pi}{3}\right)\)

\(a = 2\cos\!\left(\dfrac{\pi}{3}\right) = 2 \times \dfrac{1}{2} = 1\)
\(b = 2\sin\!\left(\dfrac{\pi}{3}\right) = 2 \times \dfrac{\sqrt{3}}{2} = \sqrt{3}\)

\(\Rightarrow z = 1 + j\sqrt{3}\)
C3

Représentation dans le plan complexe

Un nombre complexe \(z = a + jb\) se représente comme un point de coordonnées \((a, b)\) dans le plan :

• L'axe horizontal = axe réel (partie \(a\))

• L'axe vertical = axe imaginaire (partie \(b\))

\[z = a + jb \longleftrightarrow \text{Point } (a,\ b) \text{ dans le plan}\]
💡Le module \(r = |z|\) est la distance du point à l'origine. L'argument \(\theta\) est l'angle que fait le vecteur avec l'axe réel positif.
C4

Opérations sur les complexes

Addition et soustraction → forme cartésienne

\[(a+jb) \pm (c+jd) = (a \pm c) + j(b \pm d)\]
\((2-j) + (1+2j) = (2+1) + j(-1+2) = 3 + j\)

Multiplication → forme polaire ou cartésienne

En cartésienne
\[(a+jb)(c+jd) = (ac-bd) + j(ad+bc)\]

Développer comme un produit normal, puis utiliser \(j^2 = -1\)

En polaire (plus simple !)
\[r_1\text{cjs}(\theta_1) \times r_2\text{cjs}(\theta_2) = r_1 r_2\,\text{cjs}(\theta_1+\theta_2)\]

On multiplie les modules et on additionne les arguments

Exemple polaire : \(\sqrt{2}\,\text{cjs}(45°) \times \sqrt{2}\,\text{cjs}(300°)\)

\(r = \sqrt{2} \times \sqrt{2} = 2 \quad;\quad \theta = 45° + 300° = 345°\)
\(\Rightarrow 2\,\text{cjs}(345°)\)

Division → forme polaire

\[\frac{r_1\,\text{cjs}(\theta_1)}{r_2\,\text{cjs}(\theta_2)} = \frac{r_1}{r_2}\,\text{cjs}(\theta_1 - \theta_2)\]

On divise les modules et on soustrait les arguments.

Exemple : \(\dfrac{12\,\text{cjs}(330°)}{4\,\text{cjs}(210°)}\)

\(r = \dfrac{12}{4} = 3 \quad;\quad \theta = 330° - 210° = 120°\)
\(\Rightarrow 3\,\text{cjs}(120°)\)

Division en cartésienne (par le conjugué)

Multiplier numérateur et dénominateur par le conjugué du dénominateur.

\[\frac{a+jb}{c+jd} = \frac{(a+jb)(c-jd)}{(c+jd)(c-jd)} = \frac{(a+jb)(c-jd)}{c^2+d^2}\]
Exemple : \(\dfrac{1+j}{1-j}\)

\(= \dfrac{(1+j)(1+j)}{(1-j)(1+j)} = \dfrac{1 + 2j + j^2}{1 + 1} = \dfrac{1 + 2j - 1}{2} = \dfrac{2j}{2} = j\)

Puissance → forme polaire

\[\bigl(r\,\text{cjs}(\theta)\bigr)^n = r^n\,\text{cjs}(n\theta)\]
Exemple : \((3\,\text{cjs}(60°))^4 = 3^4\,\text{cjs}(240°) = 81\,\text{cjs}(240°)\)
C5

Conjugué d'un nombre complexe

Le conjugué de \(z = a + jb\) est noté \(\bar{z}\) et s'obtient en changeant le signe de la partie imaginaire.

En cartésienne
\[\bar{z} = a - jb\]
En polaire
\[\bar{z} = r\,\text{cjs}(-\theta)\]

Même module, argument opposé

Propriétés du conjugué

Produit z × z̄
\[z \cdot \bar{z} = a^2 + b^2 = r^2 = |z|^2\]

Toujours réel et positif ! Utile pour les divisions.

Somme z + z̄
\[z + \bar{z} = 2a = 2\,\text{Re}(z)\]
Conjugué d'un produit
\[\overline{z_1 z_2} = \bar{z}_1 \cdot \bar{z}_2\]
Module du conjugué
\[|\bar{z}| = |z|\]
C6

Puissances de j

Les puissances de \(j\) sont cycliques avec une période de 4.

\(j^0\)\(j^1\)\(j^2\)\(j^3\)\(j^4\)\(j^5\)\(j^6\)\(j^7\)
\(1\)\(j\)\(-1\)\(-j\) \(1\)\(j\)\(-1\)\(-j\)
💡Méthode pour \(j^n\) : diviser \(n\) par 4 et regarder le reste.
Reste 0 → \(1\)  |  Reste 1 → \(j\)  |  Reste 2 → \(-1\)  |  Reste 3 → \(-j\)

Exemple : \(j^{37} = j^{4\times9+1} = j^1 = j\)
Montrons que \(j^3 = \dfrac{1}{j}\)

\(\dfrac{1}{j} = \dfrac{1}{j} \times \dfrac{j}{j} = \dfrac{j}{j^2} = \dfrac{j}{-1} = -j = j^3\) ✓

Équations dans les complexes

Équation du 2e degré \(az^2 + bz + c = 0\)

Le discriminant peut être négatif → les solutions sont complexes !

\[\Delta = b^2 - 4ac\]
\(\Delta > 0\) — 2 solutions réelles
\[z = \frac{-b \pm \sqrt{\Delta}}{2a}\]
\(\Delta = 0\) — 1 solution double
\[z = \frac{-b}{2a}\]
\(\Delta < 0\) — 2 solutions complexes conjuguées
\[z = \frac{-b \pm j\sqrt{|\Delta|}}{2a}\]

Les deux solutions sont toujours conjuguées l'une de l'autre !

Exemple : \(4 + x^2 = 0\)

\(x^2 = -4 \implies x = \pm\sqrt{-4} = \pm 2j\)
Exemple : \(3x^2 - 4x + 2 = 0\)

\(\Delta = 16 - 24 = -8 < 0\)
\(x = \dfrac{4 \pm j\sqrt{8}}{6} = \dfrac{4 \pm 2j\sqrt{2}}{6} = \dfrac{2 \pm j\sqrt{2}}{3}\)

Équation du type \((1+j)z = -2+5j\)

Diviser les deux membres par le coefficient complexe (utiliser le conjugué).

\((1+j)z = -2+5j\)

\(z = \dfrac{-2+5j}{1+j} = \dfrac{(-2+5j)(1-j)}{(1+j)(1-j)} = \dfrac{-2+2j+5j-5j^2}{1+1} = \dfrac{-2+7j+5}{2} = \dfrac{3+7j}{2}\)

Équation du type \(z^4 + 5z^2 + 4 = 0\)

Poser \(u = z^2\) pour se ramener à une équation du 2e degré.

\(z^4 + 5z^2 + 4 = 0\) — poser \(u = z^2\)

\(u^2 + 5u + 4 = 0 \implies (u+1)(u+4) = 0\)
\(u = -1 \implies z^2 = -1 \implies z = \pm j\)
\(u = -4 \implies z^2 = -4 \implies z = \pm 2j\)

Solutions : \(z \in \{j,\ -j,\ 2j,\ -2j\}\)
