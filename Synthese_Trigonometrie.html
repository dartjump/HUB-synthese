Angles remarquables

\[\cos(\pi - x) = -\cos(x)\]
\[\sin(\pi - x) = \sin(x)\]
\[\cos\!\left(\tfrac{\pi}{2}-x\right) = \sin(x)\]
\[\sin\!\left(\tfrac{\pi}{2}-x\right) = \cos(x)\]

§5

Équations trigonométriques

sin(x) = k
\[x = \arcsin k + 2k\pi\]
\[x = \pi - \arcsin k + 2k\pi\]

2 familles de solutions

cos(x) = k
\[x = \pm\arccos k + 2k\pi\]

2 solutions (± même angle)

tan(x) = k
\[x = \arctan k + k\pi\]

1 famille (période π)

⚠️Pour \(\sin(nx)=k\) : poser \(u=nx\), résoudre, diviser par \(n\). Période → \(\frac{2\pi}{n}\) pour sin/cos, \(\frac{\pi}{n}\) pour tan.
(a) \(\sin x=\frac{1}{2}\) : \(x = \frac{\pi}{6}+2k\pi\) et \(\frac{5\pi}{6}+2k\pi\)
(b) \(\sin x=0\) : \(x = k\pi\)
(c) \(\sin x=-1\) : \(x = -\frac{\pi}{2}+2k\pi\)
(d) \(\cos x=\frac{\sqrt{2}}{2}\) : \(x = \pm\frac{\pi}{4}+2k\pi\)
(e) \(\cos x=-\frac{\sqrt{2}}{2}\) : \(x = \pm\frac{3\pi}{4}+2k\pi\)
(f) \(\sin 2x=\frac{\sqrt{3}}{2}\) : \(x = \frac{\pi}{6}+k\pi\) et \(\frac{\pi}{3}+k\pi\)
(g) \(2\cos 2x+1=0\) : \(x = \pm\frac{\pi}{3}+k\pi\)
(h) \(\sin 2x=-1\) : \(x = -\frac{\pi}{4}+k\pi\)
(i) \(\tan 3x=-1\) : \(x = -\frac{\pi}{4}+\frac{k\pi}{3} = \frac{\pi}{4}+\frac{k\pi}{3}\)
(j) \(\cos(3x+\pi)=0\) : \(x = \frac{\pi}{6}+\frac{k\pi}{3}\) (a) \(2\cos(2x+\frac{\pi}{3})=\sqrt{2}\) : \(x = \frac{17\pi}{24}+k\pi\) et \(\frac{23\pi}{24}+k\pi\)
(b) \(2\sin^2 x-\sin x-1=0\) (poser \(u=\sin x\)) : \(x = \frac{\pi}{2}+2k\pi\) et \(-\frac{\pi}{6}+2k\pi\) et \(-\frac{5\pi}{6}+2k\pi\)
(c) \(\cos^2 x=\frac{1}{4}\) : \(x = \pm\frac{\pi}{3}+k\pi\)
(d) \(1-\cos 3x=1\) : \(x = \frac{\pi}{6}+\frac{k\pi}{3}\)

Identités trigonométriques

Identité fondamentale

\[\cos^2 x + \sin^2 x = 1\]
\[\tan x = \frac{\sin x}{\cos x} \qquad \cot x = \frac{\cos x}{\sin x}\]

Double angle

\[\cos 2x = \cos^2 x - \sin^2 x = 1-2\sin^2 x = 2\cos^2 x - 1\]
\[\sin 2x = 2\sin x\cos x\]
(a) \(\sin 7a-\sin 5a = 2\sin a\cos 6a\)
(b) \(\sin u-\sin 2u+\sin 3u = \sin 2u(2\cos u-1)\)
(c) \(2\cos(-2b)\cos(3b)+2\cos(-2b)\cos(5b) = 4\cos b\cos 2b\cos 4b\)
(d) \(\cos x-\cos x\sin^2 x = \cos^3 x\)
(e) \((\sin x+\cos x)(\sin x-\cos x)+2\cos^2 x = 1\)
(f) \(\dfrac{1-\sin^4 x}{1+\sin^2 x} = \cos^2 x\) \(\cos(3a) = \cos^3 a - 3\cos a\sin^2 a\) (a) \(\sin 15° = \dfrac{\sqrt{6}-\sqrt{2}}{4}\)  ·  (b) \(\cos 75° = \dfrac{\sqrt{6}-\sqrt{2}}{4}\)  ·  (c) \(\cos 105° = \dfrac{\sqrt{2}-\sqrt{6}}{4}\)
(d) \(\cos\dfrac{3\pi}{4} = -\dfrac{\sqrt{2}}{2}\)  ·  (e) \(\sin\dfrac{11\pi}{12} = \dfrac{\sqrt{6}-\sqrt{2}}{4}\)

Formules de somme et différence

\[\cos(a\pm b)=\cos a\cos b\mp\sin a\sin b\]
\[\sin(a\pm b)=\sin a\cos b\pm\cos a\sin b\]
💡Ces formules servent à calculer \(\sin 15°\), \(\cos 75°\) etc. en décomposant l'angle : \(15°=45°-30°\), \(75°=45°+30°\)…
§8

Périodicité & forme générale \(A\sin(\omega t + \varphi)\)

\[f(t) = A\sin(\omega t + \varphi)\]
Amplitude A

valeur max de \(|f(t)|\)

Pulsation ω / Période T

\(\omega = \dfrac{2\pi}{T}\) → \(T=\dfrac{2\pi}{\omega}\)

Déphasage φ

décalage horizontal. Cos→sin : \(\varphi+\dfrac{\pi}{2}\)

💡\(\cos(x+\alpha) = \sin\!\left(x+\alpha+\dfrac{\pi}{2}\right)\) — on décale de \(+\dfrac{\pi}{2}\) pour passer de cos à sin.
(a) \(\cos\!\left(x+\frac{\pi}{4}\right) = \sin\!\left(x+\frac{3\pi}{4}\right)\)
(b) \(\cos\!\left(2x+\frac{2\pi}{3}\right) = \sin\!\left(2x+\frac{7\pi}{6}\right)\) (a) Amplitude 0,5 · période \(2\pi\) · déphasage cos nul : \(f(t)=0{,}5\sin\!\left(t+\frac{\pi}{2}\right)\)
(b) Amplitude 1 · fréquence \(\frac{1}{\pi}\) → \(\omega=2\) · déphasage cos \(\frac{\pi}{3}\) : \(f(t)=\sin\!\left(2t+\frac{5\pi}{6}\right)\)
(c) Amplitude 2 · pulsation 2 · passe par (0,0) : \(f(t)=2\sin(2t)\) (a) \(f(t)=5\sin\!\left(\dfrac{\pi}{2}t\right)\)
(b) \(f(t)=0{,}2\sin\!\left(\dfrac{\pi}{3}t-\dfrac{\pi}{3}\right)\)
(c) \(f(t)=3\sin\!\left(\dfrac{5\pi}{3}t-\dfrac{7\pi}{6}\right)\)
(d) \(f(t)=5\sin\!\left(\dfrac{2\pi}{3}t-\dfrac{6\pi}{5}\right)\)
