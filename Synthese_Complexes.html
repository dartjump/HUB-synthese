C1

Les trois représentations — rappel du cours

Un nombre complexe admet trois représentations équivalentes :

Cartésienne
\[z = a + jb\]

\(a = \text{Re}(z)\) : partie réelle
\(b = \text{Im}(z)\) : partie imaginaire

✓ Addition/soustraction

Polaire (trigonométrique)
\[z = r\,\text{cjs}(\theta)\]

\(r = |z|\) : module
\(\theta\) : argument (rad ou °)

✓ Mult./division/puissance

Exponentielle
\[z = re^{j\theta}\]

\(r = |z|\) : module
\(\theta\) : argument

≡ à la polaire, notation compacte

\[\text{cjs}(\theta) = \cos\theta + j\sin\theta\]
ConversionFormule
Polaire → Cartésienne\(a = r\cos\theta \quad b = r\sin\theta\)
Cartésienne → Polaire (module)\(r = \sqrt{a^2+b^2}\)
Cartésienne → Polaire (argument)\(\theta = \tan^{-1}\!\left(\dfrac{b}{a}\right)\) + correction quadrant
C2

Conversion Polaire / Exponentielle → Cartésienne

1

Développer : \(\text{cjs}(\theta) = \cos\theta + j\sin\theta\)

2

Calculer \(a = r\cos\theta\) et \(b = r\sin\theta\)

3

Écrire \(z = a + jb\)

\(z = 2\,\text{cjs}\!\left(\frac{\pi}{6}\right)\) → \(a=2\cos\frac{\pi}{6}=\sqrt{3}\), \(b=2\sin\frac{\pi}{6}=1\) → \(z=\sqrt{3}+j\)
\(z = 5e^{-j\frac{\pi}{3}}\) → polaire : \(5\,\text{cjs}(-\frac{\pi}{3})\) → \(a=5\cdot\frac{1}{2}=\frac{5}{2}\), \(b=5\cdot(-\frac{\sqrt{3}}{2})=-\frac{5\sqrt{3}}{2}\)
Solutions Exercice 1 — Parties réelle et imaginaire
(a) \(z=2+j6\) : \(\text{Re}=2,\ \text{Im}=6\)  ·  (b) \(z=-3j+1\) : \(\text{Re}=1,\ \text{Im}=-3\)
(c) \(z=45-j\pi\) : \(\text{Re}=45,\ \text{Im}=-\pi\)  ·  (d) \(z=-6+3j\) : \(\text{Re}=-6,\ \text{Im}=3\)
(e) \(z=\frac{4j+2}{3}\) : \(\text{Re}=\frac{2}{3},\ \text{Im}=\frac{4}{3}\)  ·  (f) \(z=\frac{-12-7j}{7}\) : \(\text{Re}=-\frac{12}{7},\ \text{Im}=-1\)
(g) \(z=2\,\text{cjs}\!\left(\frac{\pi}{4}\right)\) : \(\text{Re}=\sqrt{2},\ \text{Im}=\sqrt{2}\)  ·  (h) \(z=\frac{1}{2}\text{cjs}\!\left(\frac{\pi}{3}\right)\) : \(\text{Re}=\frac{1}{4},\ \text{Im}=\frac{\sqrt{3}}{4}\)
(i) \(z=5e^{-j\frac{\pi}{3}}\) : \(\text{Re}=\frac{5}{2},\ \text{Im}=-\frac{5\sqrt{3}}{2}\)  ·  (j) \(z=e^{j\frac{\pi}{2}}\) : \(\text{Re}=0,\ \text{Im}=1\)
C3

Conversion Cartésienne → Polaire / Exponentielle

1

Calculer \(r = \sqrt{a^2+b^2}\)

2

Calculer \(\theta = \arctan\!\left(\frac{b}{a}\right)\) — puis ajuster selon le quadrant (dessiner dans le plan !)

3

Écrire \(z = r\,\text{cjs}(\theta) = re^{j\theta}\)

⚠️Attention au quadrant !
• \(a>0\) : \(\theta = \arctan(b/a)\) (Q1 ou Q4)
• \(a<0,\ b\geq 0\) : \(\theta = \arctan(b/a)+\pi\) (Q2)
• \(a<0,\ b<0\) : \(\theta = \arctan(b/a)-\pi\) (Q3)
• \(a=0,\ b>0\) : \(\theta=\frac{\pi}{2}\)  ·  \(a=0,\ b<0\) : \(\theta=-\frac{\pi}{2}\)
\(z=-\frac{1}{2}+\frac{\sqrt{3}}{2}j\) → \(r=1\), \(\theta=\arctan(-\sqrt{3})+\pi=\frac{2\pi}{3}\) (Q2, \(a<0\)) → \(z=\text{cjs}\!\left(\frac{2\pi}{3}\right)\)
Solutions Exercice 2 — Conversions
(a) \(z=1-j\) : \(r=\sqrt{2},\ \theta=-\frac{\pi}{4}\) → \(\sqrt{2}\,\text{cjs}(-\frac{\pi}{4})=\sqrt{2}e^{-j\frac{\pi}{4}}\)
(b) \(z=1+j\) : \(r=\sqrt{2},\ \theta=\frac{\pi}{4}\) → \(\sqrt{2}\,\text{cjs}(\frac{\pi}{4})=\sqrt{2}e^{j\frac{\pi}{4}}\)
(c) \(z=\sqrt{3}+j\) : \(r=2,\ \theta=\frac{\pi}{6}\) → \(2\,\text{cjs}(\frac{\pi}{6})=2e^{j\frac{\pi}{6}}\)
(d) \(z=1-j\sqrt{3}\) : \(r=2,\ \theta=-\frac{\pi}{3}\) → \(2\,\text{cjs}(-\frac{\pi}{3})=2e^{-j\frac{\pi}{3}}\)
(e) \(z=j\) : \(r=1,\ \theta=\frac{\pi}{2}\) → \(\text{cjs}(\frac{\pi}{2})=e^{j\frac{\pi}{2}}\)
(f) \(z=-\frac{1}{2}+\frac{\sqrt{3}}{2}j\) : \(r=1,\ \theta=\frac{2\pi}{3}\) → \(\text{cjs}(\frac{2\pi}{3})=e^{j\frac{2\pi}{3}}\) ⚠ attention Q2 !
(g) \(z=5\,\text{cjs}(\frac{\pi}{3})\) → cart. : \(\frac{5}{2}(1+\sqrt{3}j)\)  ·  exp. : \(5e^{j\frac{\pi}{3}}\)
(h) \(z=2\,\text{cjs}(\frac{\pi}{6})\) → cart. : \(\sqrt{3}+j\)  ·  exp. : \(2e^{j\frac{\pi}{6}}\)
(i) \(z=e^{j\frac{\pi}{4}}\) → cart. : \(\frac{\sqrt{2}}{2}(1+j)\)  ·  pol. : \(\text{cjs}(\frac{\pi}{4})\)
(j) \(z=4e^{j\frac{5\pi}{3}}\) → cart. : \(2-2\sqrt{3}j\)  ·  pol. : \(4\,\text{cjs}(\frac{5\pi}{3})\)
C4

Représentation dans le plan complexe

Un nombre complexe \(z=a+jb\) se représente comme le point de coordonnées \((a,b)\) dans le plan :

• Axe horizontal = axe réel (partie \(a\))

• Axe vertical = axe imaginaire (partie \(b\))

• Le module \(r=|z|\) est la distance à l'origine

• L'argument \(\theta\) est l'angle avec l'axe réel positif

Exemples clés (Exercice 2)

• \(1+j\) → point \((1,1)\), Q1
• \(-\frac{1}{2}+\frac{\sqrt{3}}{2}j\) → point Q2, angle \(\frac{2\pi}{3}\)
• \(j\) → sur l'axe imaginaire positif
• \(2-2\sqrt{3}j\) → point Q4

Opérations sur les complexes

Addition et soustraction → forme cartésienne

\[(a+jb)\pm(c+jd) = (a\pm c) + j(b\pm d)\]

Multiplication, division, puissance → forme polaire

Multiplication
\[r_1\text{cjs}(\theta_1)\cdot r_2\text{cjs}(\theta_2) = r_1r_2\,\text{cjs}(\theta_1+\theta_2)\]

Modules × · Arguments +

Division
\[\frac{r_1\text{cjs}(\theta_1)}{r_2\text{cjs}(\theta_2)} = \frac{r_1}{r_2}\,\text{cjs}(\theta_1-\theta_2)\]

Modules ÷ · Arguments −

Puissance
\[(r\,\text{cjs}(\theta))^n = r^n\,\text{cjs}(n\theta)\]

Division en cartésienne — multiplier par le conjugué

\[\frac{a+jb}{c+jd} = \frac{(a+jb)(c-jd)}{c^2+d^2}\]
💡Le produit \((c+jd)(c-jd) = c^2+d^2\) est toujours réel. C'est pourquoi on multiplie par le conjugué du dénominateur !
Solutions Exercice 4
(a) \((2-j)+(1+2j) = 3+j\)
(b) \((3j+1)-2j = j+1\)
(c) \(\sqrt{2}\,\text{cjs}(45°)\cdot\sqrt{2}\,\text{cjs}(300°) = 2\,\text{cjs}(-\frac{\pi}{12})\) [somme = 345° = -15°]
(d) \(\frac{12\,\text{cjs}(330°)}{4\,\text{cjs}(210°)} = 3\,\text{cjs}(\frac{2\pi}{3})\) [330°-210°=120°]
(e) \((3\,\text{cjs}(60°))^4 = 81\,\text{cjs}(\frac{4\pi}{3})\)
(f) \((2-j)(1+2j) = 2+4j-j-2j^2 = 2+3j+2 = 4+3j\)
(g) \((2+j)^3\) : via cube remarquable \(= 2+11j\)
(h) \((j-1)^8\) : via forme polaire → \((\sqrt{2}\,\text{cjs}(135°))^8 = 16\,\text{cjs}(1080°) = 16\)
(i) \(2\,\text{cjs}(\frac{\pi}{3})+4\,\text{cjs}(\frac{5\pi}{3})\) : via cartésienne \(= 1+j\sqrt{3}+2-2j\sqrt{3} = 3-j\sqrt{3} = 2\sqrt{3}\,\text{cjs}(-\frac{\pi}{6})\)
(j) \(\frac{1+j}{1-j}\) : via conjugué \(= \frac{(1+j)^2}{2} = j\)
(k) \((\sqrt{3}+j)^3 = (2\,\text{cjs}(\frac{\pi}{6}))^3 = 8\,\text{cjs}(\frac{\pi}{2}) = 8j\)
Solutions Exercice 5 (simplification de fractions)
Méthode : multiplier numérateur et dénominateur par le conjugué du dénominateur.
(a) \(\dfrac{1}{1+j} = \dfrac{1-j}{2} = \dfrac{1-j}{2}\)
(b) \(\dfrac{j}{j-1} = \dfrac{j(-1-j) \cdot \ldots }{2} = \dfrac{1-j}{2}\)
(c) \(\dfrac{j+1}{2j} = \dfrac{1-j}{2}\)
(d) \(\dfrac{1+j}{1-j}+\dfrac{1-j}{1+j} = j + (-j) = 0\)

Conjugué d'un nombre complexe

Définition
\[\bar{z} = a - jb = r\,\text{cjs}(-\theta) = re^{-j\theta}\]

Même module, argument opposé. Symétrie par rapport à l'axe réel.

Propriétés clés
\[z\cdot\bar{z} = a^2+b^2 = r^2 = |z|^2\]
\[z+\bar{z} = 2\,\text{Re}(z) = 2a\]
Solutions Exercice 7 — conjugués des résultats de l'ex.4
(a) \(\bar{z}=3-j\)  ·  (b) \(\bar{z}=1-j\)  ·  (c) \(\bar{z}=2\,\text{cjs}(\frac{\pi}{12})\)  ·  (d) \(\bar{z}=3\,\text{cjs}(-\frac{2\pi}{3})\)
(e) \(\bar{z}=81\,\text{cjs}(-\frac{4\pi}{3})\)  ·  (f) \(\bar{z}=4-3j\)  ·  (g) \(\bar{z}=2-11j\)  ·  (h) \(\bar{z}=16\)
(i) \(\bar{z}=2\sqrt{3}\,\text{cjs}(\frac{\pi}{6})\)  ·  (j) \(\bar{z}=-j\)  ·  (k) \(\bar{z}=-8j\)

Équations dans ℂ

Équation du 2e degré avec discriminant négatif

\[\Delta = b^2-4ac < 0 \implies z = \frac{-b \pm j\sqrt{|\Delta|}}{2a}\]
💡Si \(\Delta = -k\) (négatif), alors \(\sqrt{\Delta} = j\sqrt{k}\). Les deux solutions sont toujours conjuguées l'une de l'autre !

Équation bicarrée : poser \(u = z^2\)

\[z^4+5z^2+4=0 \xrightarrow{u=z^2} u^2+5u+4=0 \implies (u+1)(u+4)=0\]

Division par un complexe

\[(1+j)z = -2+5j \implies z = \frac{-2+5j}{1+j} = \frac{(-2+5j)(1-j)}{2}\]

Solutions Exercice 8

Solutions — équations dans ℂ
(a) \((1+j)z=-2+5j\) : \(z=\dfrac{1}{2}(3+7j)\)
 Détail : \(z=\dfrac{(-2+5j)(1-j)}{2}=\dfrac{-2+2j+5j-5j^2}{2}=\dfrac{3+7j}{2}\)

(b) \(\dfrac{1}{z}=\dfrac{j}{1+j}\) → \(z=\dfrac{1+j}{j}=\dfrac{j(-1+j) \cdot \ldots}{1}\) : \(z=\sqrt{2}(1-j)\) (via forme polaire)

(c) \(4+x^2=0\) : \(x^2=-4\) → \(x=\pm 2j\)

(d) \(3x^2-4x+2=0\) : \(\Delta=16-24=-8\) → \(\sqrt{\Delta}=2\sqrt{2}j\) → \(x=\dfrac{4\pm 2\sqrt{2}j}{6}=\dfrac{2\pm j\sqrt{2}}{3}\)

(e) \(x^2-5jx=6\) : \(x^2-5jx-6=0\), \(\Delta=(-5j)^2+24=-25+24=-1\) → \(\sqrt{\Delta}=j\) → \(x=\dfrac{5j\pm j}{2}\)
 → \(x\in\{2j,\ 3j\}\)

(f) \(z^4+5z^2+4=0\) (poser \(u=z^2\)) : \(u=-1\) → \(z=\pm j\) ; \(u=-4\) → \(z=\pm 2j\)

(g) \(2z^2+5jz-3=0\) : \(\Delta=(5j)^2+24=-25+24=-1\) → \(\sqrt{\Delta}=j\) → \(z=\dfrac{-5j\pm j}{4}\)
 → \(z\in\left\{-j,\ -\dfrac{3j}{2}\right\}\)
