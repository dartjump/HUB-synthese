01 La Console DOS
Démarrage, configuration et raccourcis de la console Windows (cmd.exe)

// Démarrer la console

Méthode 1Recherche Windows
Taper cmd dans la barre de recherche Windows ou dans Exécuter (Win+R)
Win + R → cmd → Entrée
Méthode 2Explorateur
Dans la barre d'adresse de l'explorateur, taper cmd
Barre adresse → cmd → Entrée
AdminMode Administrateur
Clic droit sur cmd → Exécuter en tant qu'administrateur
Recherche → cmd → Clic droit → "Exécuter en tant qu'admin"

// Raccourcis essentiels

TAB
Autocomplétion des noms de fichiers/dossiers

Naviguer dans l'historique des commandes
Ctrl+C
Arrêter une commande en cours d'exécution

03 Fichiers & Répertoires
Créer, copier, déplacer, supprimer fichiers et dossiers

// Répertoires

MDMake Directory — Créer un dossier
Crée un ou plusieurs répertoires. Alias : MKDIR
md MonDossier mkdir C:\T1012\Labo2 Crée les dossiers intermédiaires si besoin : md ..\yolo\toto ← crée yolo ET toto
RDRemove Directory — Supprimer un dossier
Supprime un répertoire vide. Alias : RMDIR. Utiliser /S pour tout supprimer.
rd MonDossier ← vide seulement rd MonDossier /S ← avec tout le contenu rd MonDossier /S /Q ← sans confirmation
CDChange Directory — Naviguer
Se déplacer dans l'arborescence
cd NomDossier ← descendre cd .. ← remonter d'un niveau cd \ ← aller à la racine cd C:\Users\moi ← chemin absolu

// Fichiers

COPYCopier des fichiers
Copie un ou plusieurs fichiers vers une destination
copy source.txt dest.txt copy *.txt D:\backup\ copy C:\*.txt D:\ ← depuis C: vers D:
MOVEDéplacer des fichiers
Déplace ou renomme un fichier
move fichier.txt D:\ move ancien.txt nouveau.txt ← renommer
DELDelete — Supprimer des fichiers
Supprime un ou plusieurs fichiers. Alias : ERASE
del fichier.txt del *.tmp ← tous les .tmp del *.* /Q ← sans confirmation
RENRename — Renommer
Renomme un fichier ou répertoire. Alias : RENAME
ren vieux.txt nouveau.txt ren *.txt *.texte ← changer extension
XCOPYCopie avancée (avec sous-dossiers)
Copie fichiers et arborescences entières
xcopy source dest /S /E /I
ATTRIBAttributs de fichiers
Modifier les attributs : R=lecture seule, H=caché, A=archive
attrib +R fichier.txt ← lecture seule attrib -H fichier.txt ← enlever caché

// Redirection & Opérateurs

>
Rediriger la sortie vers un fichier (écrase)
dir > liste.txt
>>
Rediriger et ajouter à la fin du fichier
dir >> liste.txt
|
Pipe — envoyer la sortie vers une autre commande
dir | more
&
Exécuter deux commandes l'une après l'autre
cls & dir
&&
Exécuter la 2e commande seulement si la 1re réussit
mkdir test && cd test

04 Commandes Réseau

// PING

Teste la connectivité réseau vers une adresse IP ou un nom de domaine. Envoie des paquets ICMP et mesure le temps de réponse (TTL).

05 Batch & Variables
Fichiers .bat — automatiser des séquences de commandes DOS

// Qu'est-ce qu'un fichier batch ?

📄 Définition Un fichier .bat (ou .cmd) est un fichier texte contenant une suite de commandes DOS qui s'exécutent automatiquement dans l'ordre. C'est la base de l'automatisation sous Windows.

// Instructions de base dans un batch

@ECHO OFFMasquer l'affichage des commandes
Mettre en première ligne pour ne pas afficher chaque commande exécutée
@echo off REM les commandes s'exécutent sans s'afficher
REMCommentaires
Ajouter des commentaires dans le batch (non exécutés)
REM Ceci est un commentaire REM Auteur : Liam REM Date : 2025
ECHOAfficher un message
Afficher du texte dans la console
echo Bonjour ! echo. ← ligne vide echo Fin du script
PAUSEAttendre une touche
Suspendre l'exécution jusqu'à ce que l'utilisateur appuie sur une touche
pause REM affiche "Appuyez sur une touche..."
COLORChanger la couleur de la console
Code : 2 chiffres hex — fond puis texte. 0=noir, F=blanc, A=vert, E=jaune…
color F0 ← fond blanc, texte noir color 0A ← fond noir, texte vert color 1E ← fond bleu, texte jaune
PROMPTChanger l'invite de commande
$D=date, $T=heure, $P=chemin, $G=signe >
prompt $D $P$G → 03/03/2026 C:\T1012>

// Variables d'environnement

Les variables sont entourées de %. On les utilise avec SET.
REM Afficher toutes les variables : set REM Définir une variable : set MONVAR=bonjour echo %MONVAR% REM Variables prédéfinies importantes : echo %USERPROFILE% ← C:\Users\Liam echo %USERNAME% ← Liam echo %HOMEDRIVE% ← C: echo %HOMEPATH% ← \Users\Liam echo %TEMP% ← dossier temporaire echo %PATH% ← chemins des exécutables echo %COMPUTERNAME% ← nom du PC

// Modifier le PATH

⚠ Attention La modification du PATH dans un batch n'est valide que pour la console en cours. À la fermeture, le PATH initial est restauré. Pour une modification permanente : Propriétés système → Variables d'environnement.
REM Ajouter un chemin au PATH existant : set PATH=%PATH%;%HOMEDRIVE%%HOMEPATH%\Mes documents\Mes Batch REM Vérifier : echo %PATH%

06 Labo 5 — VMware & Batch avancé
Partages réseau, lecteurs réseau, batch automatique au démarrage
💡 Prérequis Le labo 5 se fait dans une machine virtuelle VMware. Session administrateur avec mot de passe EphecEphec. Créer un dossier partage dans Mes documents et le partager.

// 5.B.2 — Batch : connecter deux lecteurs réseau

connecter.bat
@echo off REM Connexion de deux lecteurs réseau net use X: \\ip1\partage net use Y: \\ip2\partage echo Lecteurs X: et Y: connectés ! pause
Vérifier dans la console : net use Vérifier dans l'explorateur → Ce PC → lecteurs X: et Y: visibles

// 5.B.3 — Batch : supprimer les lecteurs réseau

deconnecter.bat
@echo off REM Suppression des lecteurs réseau X: et Y: net use X: /delete net use Y: /delete echo Lecteurs supprimés. pause

// 5.B.4 — contenu.bat — lister les fichiers dans un .txt

contenu.bat
@echo off REM Crée contenu.txt dans le profil utilisateur REM avec uniquement les noms de fichiers du répertoire courant dir /b > %USERPROFILE%\contenu.txt echo Fichier contenu.txt créé dans %USERPROFILE% pause
💡 Option /b de DIR dir /b affiche uniquement les noms des fichiers (format "bare"), sans informations supplémentaires.

// 5.B.5 — Batch qui ajoute un dossier au PATH

setpath.bat
@echo off REM Ajoute "Mes Batch" au PATH en utilisant HOMEDRIVE et HOMEPATH set PATH=%PATH%;%HOMEDRIVE%%HOMEPATH%\Mes documents\Mes Batch echo PATH mis à jour. echo Nouveau PATH : %PATH% pause
